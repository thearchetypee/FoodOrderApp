# FoodOrderApp
This is a sample project to to learn React. It uses Context API, states, hooks. For BE I am using Firebase realtime database

## Running Locally
```bash
git clone https://github.com/aayush287/FoodOrderApp.git
cd FoodOrderApp
npm install
npm start
```
